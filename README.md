WhatsApp Outreach Automator
Features

Excel Integration: Load dynamic contact lists and custom message parts directly.

Multimedia Support: Automatically attach images or videos based on the file path in Excel.

Session Persistence: Keeps you logged in between runs using a local Chrome profile.

Contact Sync: Export existing chat lists from WhatsApp to Excel.

Setup

Install dependencies: pip install -r requirements.txt

Prepare contacts.xlsx with columns: Number, Name, Tailored_Text, Media_Path.

Run python main.py.

Scan the QR code on the first run.
